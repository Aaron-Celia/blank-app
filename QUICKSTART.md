# Quick Start Guide

## Installation & First Run

```bash
# Run the trainer
python blackjack_trainer.py
```

## 5-Minute Walkthrough

### 1. Try the Speed Count Drill (1 minute)
```
Main Menu → 1 (Card Counting Training)
         → 1 (Single Card Drill)
         → 10 (number of cards)
```
This shows you how the Hi-Lo system works. Count each card:
- 2-6: +1
- 7-9: 0
- 10-A: -1

### 2. Test Your Basic Strategy (2 minutes)
```
Main Menu → 2 (Basic Strategy Training)
         → 1 (Quiz Mode)
         → 5 (number of questions)
```
See how well you know basic strategy. The app will tell you the correct answer.

### 3. Play a Practice Game (2 minutes)
```
Main Menu → 3 (Full Game Simulator)
         → 10000 (starting bankroll)
         → y (show hints)
```
Play a few hands and watch how counting works in real time.

## Learning Path

1. **Week 1-2:** Basic Strategy Quiz until 95%+ accuracy
2. **Week 3-4:** Single Card Drills until you can count fast
3. **Week 5-6:** Running Count Drills + True Count Conversion
4. **Week 7-8:** Full Game Simulator with counting

### Path 2: Know Basic Strategy (4 weeks)
1. **Week 1-2:** Speed count drills daily
2. **Week 3:** True count conversion practice
3. **Week 4:** Full game simulator

### Path 3: Practice Before Casino Trip (Quick)
1. Review basic strategy chart (Menu → 7)
2. Practice 50 hands in full game mode
3. Review bankroll management (Menu → 4)

## Quick Reference

### Hi-Lo Values
```
+1: 2 3 4 5 6
 0: 7 8 9
-1: 10 J Q K A
```

### Betting Spread
```
TC ≤ +1:  Min bet
TC = +2:  2-3 units
TC = +3:  4-5 units
TC = +4:  6-8 units
TC ≥ +5:  8-12 units
```

### Critical Basic Strategy
```
Always Split:    8,8 and A,A
Never Split:     10,10 and 5,5
Double on 11:    (except vs A in S17)
Surrender:       16 vs 9,10,A
                 15 vs 10
```
